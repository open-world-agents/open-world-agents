Database Setup Instructions

If you haven't created the database yet, follow these steps:

  1. Run the analysis script: 
    python analyze_datasets.py --dataset-root /path/to/dataset --db-path dataset_analysis.db
  2. Verify the database was created: 
    ls -la dataset_analysis.db
  3. Start the web server: 
    cd website && python app.py
Note: The web application expects the database file to be located at ../dataset_analysis.db relative to the website directory.
